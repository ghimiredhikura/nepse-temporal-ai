# Results

- `paper/`: compact numerical artefacts supporting the reported evaluation,
  inference, calibration, and explanation results.
- `figures/`: final empirical figures used in the manuscript. The conceptual
  method-workflow figure is not part of the code and data release.
- `summary/`: one headline table and figure regenerated by `python main.py`.
- `generated/`: ignored output location for rerun experiments.

Runtime logs, large row-level prediction files, and trained model checkpoints
are not included.
